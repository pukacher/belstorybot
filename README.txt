📘 MyBelstory Bot — Инструкция по запуску

Этот Telegram-бот проверяет подписку на канал и выдает PDF-гайд "Капсула для отпуска".

──────────────────────────────
⚙️ Что внутри архива:
- bot.py — код бота
- Гайд Капсула для отпуска.pdf — сам файл для выдачи пользователям
──────────────────────────────

🚀 Как запустить бота:

1️⃣ Установите Python 3.10 или новее:
   https://www.python.org/downloads/

2️⃣ Установите библиотеку aiogram:
   Откройте терминал или командную строку в папке с ботом и выполните:
   pip install aiogram==3.*

3️⃣ Запустите бота:
   python bot.py

4️⃣ Перейдите в Telegram, найдите своего бота (созданного через @BotFather) и отправьте команду:
   /start

──────────────────────────────
💬 Как работает бот:
- Приветствует пользователя.
- Просит подписаться на канал @BretelkaSalon.
- Проверяет подписку.
- Если подписка есть — выдает гайд.
- Если нет — предлагает подписаться еще раз.
